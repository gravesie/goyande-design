# Goyande brand

Brand definition and assets for Goyande. This is the source of truth for the
logo, colours, typography and supporting graphics.

## Start here

- **`Goyande-Brand-Guidelines-V0-3.html`** — the brand guidelines. Open in a
  browser. Covers the logo, the navy/teal palette, Manrope typography, the
  gradient, usage rules and the PGGI co-brand lockup.

## assets/

Source SVGs — the editable, scalable originals.

- `goyande-mark*.svg` — the lotus mark (colour, mono navy, white, reversed)
- `goyande-lockup-horizontal.svg`, `goyande-lockup-stacked.svg` — logo lockups
- `goyande-icon.svg`, `goyande-favicon.svg` — app/icon forms
- `goyande-gradient-hero.svg` — the brand gradient
- `goyande-logo-pggi-pair.svg`, `goyande-pggi-cobrand.svg` — PGGI co-branding
- `goyande-proof-*.svg`, `goyande-card-*.svg` — supporting graphics
- `goyande-slide-*.svg`, `goyande-banner-linkedin.svg`, `goyande-social-card.svg`,
  `goyande-businesscard-back.svg` — applied templates

## exports/

Rendered PNGs and favicons for direct use where SVG isn't practical (decks,
social, the icon at fixed sizes). Same names as the SVG sources.

## Notes

- Manrope is the brand typeface (open source, SIL Open Font License).
- Edit the SVGs in `assets/`; treat `exports/` as generated output.
